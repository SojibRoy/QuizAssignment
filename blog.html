1. What is the main difference between local storage and session storage?

Local Storage

  • The storage capacity of local storage is 5MB/10MB.
  • There are fewer old browsers that support it.
  • The client can only read local storage.
  • There is no transfer of data to the server.

Session Storage

  • The storage capacity of session storage is 5MB.
  • There are fewer old browsers that support it.
  • The client can only read local storage.
  • There is no transfer of data to the server.

2. What is the main difference between Global Scope and Block Scope?

Global Scope

  • A variable declared outside a function, becomes GLOBAL.
  • Global variables can be accessed from anywhere in a JavaScript program.
  • If you assign a value to a variable that has not been declared,is a Global variable.

Block Scope

  • Variables declared inside a { } block cannot be accessed from outside the block.
  • Let & const are also called as block scope that is they are accessible within that particular block.
  • A block scope will be the same as a function scope.
